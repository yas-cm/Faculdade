Criar uma aplicação em Java para gerenciar uma lista de
empregados em uma empresa, onde cada empregado tem
um nome, idade e um cargo representado por uma letra
('G' para Gerente, 'D' para Desenvolvedor, e 'A' para
Analista).

A aplicação deve permitir a adição, remoção e atualização
das informações dos empregados. Além disso, a aplicação
deve fornecer a capacidade de filtrar e imprimir apenas os
empregados que possuem um cargo específico.

Se o cargo filtrado não estiver entre 'G', 'D', ou 'A', a
aplicação deve retornar a mensagem "Cargo não
encontrado".

--------------- Requisitos da aplicação ------------------
- Criar uma classe Empregado que contenha os atributos
nome, idade, e cargo.
- Implementar métodos de acesso (get) para cada um dos
atributos.
- Criar uma classe ListaEmpregados que gerencie uma lista
de objetos Empregado utilizando a classe ArrayList. ("classe cliente", uma lista de 
cliente que vai ter objetos relacionados a empregado)

A classe ListaEmpregados deve conter métodos para:
- adicionar um empregado;
- remover um empregado com base no índice; (0,1,2,3 e tals)
- atualizar as informações de um empregado com base no
índice;
- imprimir a lista completa de empregados:
	um método deve ser utilizado para retornar o nome
	completo do cargo conforme 'G', 'D' ou 'A' (Gerente,
	Desenvolvedor ou Analista);
- filtrar e imprimir a lista de empregados com base em um
cargo específico ('G', 'D', 'A'); caso o cargo seja diferente,
exibir "Cargo não encontrado".

No método main, demonstrar o funcionamento dessas
funcionalidades, incluindo:
- a adição de pelo menos três empregados;
- a remoção de um empregado;
- a atualização de informações de um empregado;
- a filtragem de empregados por cargo
